# Security Policy

## Supported versions

The default branch is the supported version of this content repository.

## Reporting a vulnerability

Do not open a public issue for security-sensitive reports. Contact the repository maintainers privately with:

- a description of the issue
- affected files or workflows
- reproduction steps, if available
- any suggested mitigation

Maintainers will acknowledge the report, evaluate impact, and coordinate a fix before public disclosure when needed.
