# Code of Conduct

This project follows the [Contributor Covenant Code of Conduct](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Our pledge

We pledge to make participation in this project respectful, welcoming, and harassment-free for everyone.

## Expected behavior

- Use welcoming and inclusive language.
- Respect differing viewpoints and experiences.
- Accept constructive feedback gracefully.
- Focus on what is best for learners and maintainers.

## Unacceptable behavior

- Harassment, intimidation, or discriminatory language.
- Personal attacks or sustained disruption.
- Publishing private information without explicit permission.

## Enforcement

Report concerns through the repository maintainers. Maintainers may remove comments, close issues, or restrict participation when needed to protect the community.
